# Warung Sayur (Flutter)

Paket ini berisi folder `lib/` + `test/` + `pubspec.yaml` untuk aplikasi Warung Sayur.

## Cara pakai
1) Buat project Flutter baru:
```bash
flutter create warung_sayur
cd warung_sayur
```
2) Ganti file/folder berikut dengan yang ada di zip ini:
- `pubspec.yaml`
- `analysis_options.yaml`
- folder `lib/`
- folder `test/` (opsional)

3) Install dependency:
```bash
flutter pub get
```

4) Jalankan:
```bash
flutter run
```

5) Jalankan test:
```bash
flutter test
```

## Backup / Restore
Menu **Backup**:
- **Buat Backup (.json)**: membuat file backup lalu share (WA/Drive).
- **Restore dari Backup (.json)**: pilih file backup lalu data dipulihkan (overwrite).

## Export CSV
Menu **Backup** juga punya **Export & Share CSV** untuk ringkasan per tanggal.
